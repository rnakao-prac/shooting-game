# 🚀 スペースシューター

AIを使って作ったブラウザで遊べるシューティングゲームです。

## 遊び方

`index.html` をブラウザで開くだけで即プレイできます。

| キー | 操作 |
|---|---|
| ← → ↑ ↓ | 自機を移動 |
| スペース | 弾を発射 |

## 敵の種類

| 敵 | 色 | 特徴 | スコア |
|---|---|---|---|
| ノーマル | 赤 | 標準速度 | 10pt |
| ファスト | 橙 | 高速移動 | 20pt |
| タンク | 紫 | HP3・低速 | 50pt |

## ゲームの特徴

- スコアが上がるほどレベルアップ（敵が増加・高速化）
- 爆発エフェクト・星スクロール背景
- ライフ制（3回被弾でゲームオーバー）

## 技術スタック

- HTML / CSS / JavaScript（フレームワーク不使用）
- Canvas API でゲームを描画

## このプロジェクトの目的

Claude Code（AI）を使ったバイブコーディングと、  
Git / GitHub を使ったコード管理の学習用プロジェクトです。
